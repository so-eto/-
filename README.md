# 前株後株判断アプリ (Company Name Formatter)

入力された会社名から「前株（株式会社〇〇）」や「後株（〇〇株式会社）」を瞬時に判別し、名表記のブレを統一・整形する簡易ユーティリティツールです。

営業活動、CA（キャリアアドバイザー）の求職者管理、求人データベースの登録時における「企業名入力の表記揺れ」を防ぎ、データクレンジングの作業を効率化するために開発されました。

---

## 🚀 特徴 (Features)

- **自動判別機能**: 企業名に含まれる「株式会社」「有限会社」などの位置を自動検出。
- **ブレ表記の補正**: 「(株)」「株)」「（株）」などの略称表記を、正式な「株式会社」に一瞬で置換・統一。
- **データ登録の効率化**: コピペするだけで、データベースやスプレッドシートへの登録に適したクリーンなテキストを出力。

---

## 🛠️ 技術スタック (Tech Stack)

このプロジェクトは以下の技術または環境で動かすことを想定しています：

- **Language**: JavaScript / Python
- **Platform**: Web Browser (HTML/JS) または Google Apps Script (GAS)
- **Database Link**: Google Spreadsheet等とのAPI連携（予定）

---

## 📖 使い方 (Usage)

### Web版（またはコンソール）の場合
1. 入力欄に任意のテキスト（例：`（株）レバテック` や `メルカリ 株`）を入力します。
2. 実行ボタンを押すと、以下のルールに従って自動整形されたテキストが出力されます。

```text
[入力] （株）レバテック  -->  [出力] 株式会社レバテック（前株）
[入力] メルカリ 株       -->  [出力] 株式会社メルカリ（後株）
